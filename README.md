# Fake Sober｜白天人格測驗 ☀️🥐

「如果 Fake Sober 的麵包是一種人，你會是哪一種？」

## 開發

```bash
npm install
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000)

## 部署（Vercel）

專案根目錄即 Next.js 應用（`package.json` 在 repo 根目錄）。

- **Root Directory**：留空或 `.`
- **Framework Preset**：Next.js

若仍使用舊子目錄結構，請將 Root Directory 設為 `fakesober_bread_test`（已不再需要）。
